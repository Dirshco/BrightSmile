🚀 Step-by-Step: Upload to Google Play Store

  1. Create developer account at play.google.com/console ($25 one-time fee).
  2. Set up tax information (W-9 or local equivalent).
  3. Click "Create app" → default language English, app name: "Oral Health Tracker - Dental Care".
  4. Prepare store listing – upload screenshots, feature graphic, icon, short/long description (use texts above).
  5. Complete content rating – use IARC answers from the table (all “None” / “No”).
  6. Add privacy policy URL – paste your GitHub Pages link (hosted HTML).
  7. Set “Pricing & distribution” → Free, all countries selected.
  8. Upload app bundle (AAB) – Build → Generate Signed Bundle / APK → Android App Bundle. Use your keystore.
  9. Fill release notes (English & Arabic – see below).
  10. Review & rollout → “Start rollout to Production”. Wait 2–6 hours for review.

🛡️ Data Safety Section (Play Console)

When filling "Data safety" form in Play Console, select:

  • ✅ No data collected or shared
  • ✅ No user data is collected
  • ✅ All data is local and encrypted (Android sandbox)
  • ✅ No third-party libraries that collect data
  • ✅ Security practices: data is stored only on device, no transmission

💡 Tip: Because the app uses no analytics and no crashlytics, you can honestly declare "No data collected".
